About

We in DreamOn Augmented Reality are trying to make our life much more easier by using Augmented Reality. We use Vuforia and Unity for development. Unity is a game engine and Vuforia is a library made by Qualcomm used to implement augmented reality and provides SDK for Android development, IOS, and Unity. Using these technologies we are trying to make out a best from better and give user a feel of virtual object in real world

An application of augmented reality powered by Unity & Vuforia. This app is use to analyze solar system in real world virtually. Basically a user can watch whole solar system with the help of mobile or any device in real world. This app uses camera and textures (images). Camera focuses on a texture and respective model generated and a user can easily watch that model by in real world. To use this app, download below zip file which has textures for solar system.
